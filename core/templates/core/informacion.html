Información sobre el proyecto

🧠 Sobre este proyecto

Este test ha sido desarrollado como parte del Trabajo de Fin de Grado (TFG) de un alumno de la ETSIDI, escuela de la UPM, con el objetivo de explorar el estado emocional y el nivel de ansiedad de los usuarios.

⚠️ Importante:
Este cuestionario no tiene validez diagnóstica y no sustituye la evaluación de un profesional de la salud mental. Las respuestas recogidas ayudan a ofrecer una orientación general sobre el nivel de tensión o ansiedad percibido en el momento, pero no deben interpretarse como un diagnóstico clínico.

📊 ¿Qué mide este test?

Este test se basa en el test de ansiedad estado, ampliamente utilizado en psicología para medir el nivel de ansiedad actual, mediante preguntas sobre tranquilidad, preocupación, descanso, entre otros aspectos.

Al finalizar, se calcula una puntuación total que se interpreta de forma orientativa, mostrando un mensaje general según intervalos definidos.

🔒 Privacidad

Los datos introducidos no se comparten con terceros y se utilizan exclusivamente para fines de este proyecto.
